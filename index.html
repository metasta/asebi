馬酔木明朝は旧字明朝体フォントです。「IPAex明朝」（正確にはその派生プログラム「三番明朝」）に収録された漢字のうち、新字や俗字、誤字ほか約二千字の字形を「IPAmj明朝」収録の旧字形に変更しました。その際『新字源 改訂版』（角川書店、第24版、1997）の掲げる字形および記述を判断の基準としました。また、かな文字および約やく物ものには「錦明朝かな」の文字を使用しました。「っ」「ゃ」などの捨てがなはカタカナでは小さく、ひらがなでは通常のかなとおなじ大きさにしてあります。

フォント作成に関する情報や改訂履歴は GitHub に公開してあります。
